# Uni Model Builder

Attempt to build a model of students at a University based on bits and pieces of information.

## Setup

### Venv

#### Windows

```bash
python -m venv ./venv/
./venv/Scripts/activate
```

#### Linux

```bash
python3 -m venv ./venv/
source venv/bin/activate
```
